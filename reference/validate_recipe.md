# Check valid recipe

Check valid recipe

## Usage

``` r
validate_recipe(svy_type, svy_edition, recipe_svy_edition, recipe_svy_type)
```

## Arguments

- svy_type:

  Type of survey

- svy_edition:

  Edition of the survey

- recipe_svy_edition:

  Edition of the recipe

- recipe_svy_type:

  Type of the recipe

## Value

Logical
