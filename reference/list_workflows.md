# List all workflows

List all workflows from the active workflow backend.

## Usage

``` r
list_workflows()
```

## Value

List of all RecipeWorkflow objects.

## See also

[`search_workflows`](https://metasurveyr.github.io/metasurvey/reference/search_workflows.md),
[`filter_workflows`](https://metasurveyr.github.io/metasurvey/reference/filter_workflows.md)

## Examples

``` r
if (FALSE) { # \dontrun{
all <- list_workflows()
} # }
```
