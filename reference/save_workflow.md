# Save a RecipeWorkflow to a JSON file

Save a RecipeWorkflow to a JSON file

## Usage

``` r
save_workflow(wf, file)
```

## Arguments

- wf:

  A RecipeWorkflow object

- file:

  Character file path

## Value

NULL (called for side-effect)

## Examples

``` r
if (FALSE) { # \dontrun{
save_workflow(wf, "my_workflow.json")
} # }
```
