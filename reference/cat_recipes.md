# cat_recipes

Cast recipes from survey

## Usage

``` r
cat_recipes(self)
```

## Arguments

- self:

  Object of class Survey

## Value

Character string listing recipe names, or "None".

## Examples

``` r
if (FALSE) { # \dontrun{
svy <- survey_empty(type = "demo", edition = "2023")
cat_recipes(svy)
} # }
```
